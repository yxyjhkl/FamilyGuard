# 家庭保障检视 (FamilyGuard)

一款面向保险业务员的家庭保障检视手机应用，帮助快速构建家庭结构、录入成员保障信息、直观展示保障缺口，并生成精美的保障检视图分享给客户。

## 技术栈

| 层级 | 技术 |
| --- | --- |
| 框架 | React Native 0.76 + TypeScript |
| 导航 | React Navigation 6 (Native Stack) |
| 存储 | AsyncStorage (本地持久化) |
| UI | React Native Paper (Material Design) |
| 图片导出 | react-native-view-shot |
| 分享 | react-native-share + CameraRoll |
| 图标 | react-native-vector-icons |

## 核心功能

### 1. 家庭结构选择（20+ 种预置模板）

**常见家庭**：二人世界、夫妻+1子、夫妻+1女、标准四口之家、二胎家庭、三孩家庭、三代同堂

**多元家庭**：丁克家庭、独居一人、单亲爸爸+子女、单亲妈妈+子女、丧偶父亲+子女、丧偶母亲+子女、重组家庭（双方带孩/男方带孩/女方带孩）、隔代家庭（祖孙）、空巢老人、独居单亲+成年子女

### 2. 保障录入与缺口分析

- **7项保障**：重疾、身价、医疗、养老、防癌、意外、教育
- **7项权益**：信托、国医、居养、家医、特药、超疗、税优（含有效期备注）
- 自动计算保障缺口

### 3. 导出与分享

- 精美检视图片（金句 + 保障总览 + 成员详情）
- 隐私控制：可隐藏姓名和关键信息
- 业务员名片开关：显示/隐藏姓名和电话
- 支持保存到相册和微信分享
- 时间戳 + 金句 + 版权声明

### 4. AI总结（预留接口）

- 预留 `aiService.ts`，后期接入 API Key 后自动生成智能总结

## 快速开始

```bash
# 1. 安装依赖
cd FamilyGuard
npm install

# 2. iOS 安装原生依赖
cd ios && pod install && cd ..

# 3. 启动
npx react-native start

# 4. 运行（Android/iOS）
npx react-native run-android
npx react-native run-ios
```

## 使用流程

1. **创建家庭** → 选择家庭结构模板 → 输入家庭名称
2. **编辑成员保障** → 为每个成员配置保障项和权益
3. **导出检视图** → 配置隐私选项 → 选择金句 → 保存相册 / 微信分享
